# Personal color palette

Color palette used for theming https://wjakethompson.com. The palette
also includes color scales that can be used in place of
[`ggplot2::scale_color_discrete()`](https://ggplot2.tidyverse.org/reference/scale_colour_discrete.html)
or
[`ggplot2::scale_fill_discrete()`](https://ggplot2.tidyverse.org/reference/scale_colour_discrete.html).

## Usage

``` r
palette_wjake
```

## Format

A character vector of 5 hex color codes.

## Examples

``` r
scales::show_col(palette_wjake)
```
